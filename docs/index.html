Running in three minutes

Open source. Build it yourself, own it forever.

Add your key

Paste an Anthropic or OpenAI API key in Settings. It goes straight into your iPhone's Keychain — nowhere else. Either key alone works.

Questions, answered

What is SuperSiri?

An open-source AI assistant app for iPhone that combines Anthropic's Claude and OpenAI's GPT models in one place — and gives them the ability to act: manage your calendar and reminders, look up contacts, find places, control HomeKit devices, remember facts about you, and search the web.

Which AI models does it support?

Claude Opus 4.8, Claude Sonnet 5, Claude Haiku 4.5, GPT-5.1, and GPT-5.1 mini. Switch mid-conversation from the toolbar — the model catalog is one Swift file, so adding new releases takes a minute.

Is it free?

The app is free and open source. You bring your own API key from Anthropic or OpenAI and pay those providers only for what you use — no subscription, no markup, no middleman.

How private is it?

API keys are stored in the iOS Keychain and sent only to the provider you chose. Conversations, workflows, and memory never leave your device. There's no analytics SDK and no SuperSiri backend — the architecture makes tracking impossible, not just unlikely.

Does it replace Siri?

Apple doesn't allow replacing Siri outright, but SuperSiri gets remarkably close: “Ask SuperSiri” routes questions from Siri to Claude or GPT, Voice Mode gives you continuous hands-free conversation, and every workflow is a Shortcuts action you can wire to Focus modes, automations, or the Action Button.

What can it actually automate?

In agent mode the AI acts directly — creating events, setting reminders, switching lights, finding contact details, searching places and the web, chaining several tools in one request. Workflows go further: multi-step AI pipelines (summarize → draft, critique → plan) with a different model per step, runnable by voice.
